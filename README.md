# Global AI Language Translator 🌍

This is a simple and efficient Language Translator web application built using **Python** and **Streamlit**. It uses the `deep-translator` library to provide accurate translations across multiple languages.

## Features
- **Auto-detection:** Detects the source language automatically.
- **Multiple Languages:** Supports translation into various languages like Hindi, Punjabi, German, etc.
- **User-Friendly UI:** Built with Streamlit for a clean and interactive experience.

## Tech Stack
- Python
- Streamlit
- Deep-translator

## How to Run
1. Install requirements: `pip install streamlit deep-translator`
2. Run the app: `streamlit run app.py`
